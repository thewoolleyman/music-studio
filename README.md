# Music Studio Documentation

This repository is a local working set for the music studio setup. It keeps device manuals, installers, saved vendor pages, media assets, and setup notes together by hardware folder.

The current studio plan centers on using Logic with a Focusrite Scarlett 18i20 4th Gen as the single Mac audio and MIDI interface. The Roland V-Drums V51 feeds the 18i20 on rear inputs 1/2, the Akai MPC One Plus feeds rear inputs 3/4 and uses the 18i20 MIDI In/Out loop, and monitoring is through 18i20 Headphone out 1 with AIAIAI TMA-2 Studio Wireless headphones.

For the detailed device roles, connection diagram, tested stereo capture path, and troubleshooting rules, see [AGENTS.md](AGENTS.md).
