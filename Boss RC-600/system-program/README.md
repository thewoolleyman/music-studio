# Boss RC-600 System Program

## Current Archive

- `rc600_sys_v150.zip`: RC-600 System Program 1.50, downloaded from the official BOSS RC-600 Updates & Drivers page.

## Before Updating

- Check the version already installed on the RC-600 before applying this archive.
- Do not apply this update data to any product other than the Boss RC-600.
- Do not power off the RC-600 while the update is writing.

## Version Check

1. Hold `MENU`.
2. Press the `POWER` switch.
3. Read the system version shown on the display.
4. Power the unit off.

## Update Outline

1. Extract `rc600_sys_v150.zip`.
2. Hold `EXIT`.
3. Press the `POWER` switch to boot the RC-600 in update mode.
4. Connect the RC-600 to the Mac over USB.
5. Copy all files from the extracted `rc600_sys_v150` folder to the mounted `RC-600` drive.
6. Eject the `RC-600` drive from macOS.
7. Press `ENTER` on the RC-600 to run the update.
8. Wait for `COMPLETE`, then power the unit off.
